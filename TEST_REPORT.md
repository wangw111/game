## 部署说明

### 快速启动
```bash
# 1. 进入项目目录
cd game

# 2. 运行启动脚本
./start.sh

# 3. 访问应用
# 浏览器打开: http://localhost:8501
```

### 手动启动
```bash
# 1. 创建虚拟环境
python3 -m venv venv

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.example .env
# 编辑.env文件，添加API密钥

# 5. 启动应用
streamlit run app.py
```
